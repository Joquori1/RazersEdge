Project
Razors Edge Data Visualization

Our front end, data visualization web application leverages charting libraries to build charts, graphs and maps using datasets pulled from a series of APIs. As part of our collaboration workflow, we used GitHub Project Boards as a SCRUM framework for tracking project tasks from inception to completion. Progressive iterations of code commits were pushed up to branches on our GitHub repository and reviewed priot to merging to main.

APIs

Where the ISS At? - Real-Time ISS Location
Source: https://api.wheretheiss.at/v1/satellites/25544
Usage: Live data feed for charting location of the International Space Station

Forbes 400 Rich List - A Real-Time API
Source: https://forbes400.herokuapp.com/api/forbes400
Usage: Dataset for charting highest net worth individuals

NCHS* Historical Life Expectancy of US population
Source: https://data.cdc.gov/resource/w9j2-ggv5.csv
Usage: Dataset for charting US population life expectancy
* NCHS(National Center for Health Statistics)

NASA Combined Land-surface, Air & Sea-surface Water Temperature from 1951 to 1980
Source: https://data.giss.nasa.gov/gistemp/tabledata_v4/GLB.Ts+dSST.csv
Usage: Dataset for charting global temperature
* NASA(National Aeronautics and Space Administration)
